# Dataprocessing
**Bio-Informatics Year 3, Period 11 (2022-2023)**  
This repository contains code and data for excersises and the final assignment of the Dataprocessing course from year 3 of the Bio-Informatics Bachelor program at the Hanze University of Applied Sciences. The goal of this course is to gain experience creating data analysis workflows using snakemake, bash, and python scripts.
